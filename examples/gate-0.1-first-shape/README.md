# Gate 0.1: first shape on screen

Proves the general spine end to end: a hand-authored feature-tree document builds
through the pure executor onto the kernel, exports glTF, and renders in the viewer.

```bash
ncad build examples/gate-0.1-first-shape/block.hocon
ncad
```

`block.hocon`: a rectangle sketched on XY, extruded into an 80 x 60 x 8 mm block.
