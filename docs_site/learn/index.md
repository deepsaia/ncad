# Learn: an engineering reference

A product-neutral document across the subjects ncad touches: **Foundations** (math and geometry),
**Geometric and Solid Modeling**, **Manufacturing (CAM)**, **Electronic Design (PCB / ECAD)**,
**Kinematics and Mechanisms**, **Multibody Dynamics**, **Robotics and Physics-Engine
Simulation**, **Engineering Analysis (CAE)**, and **Interchange and Standards**.

Every concept is taught to professional depth whether or not ncad implements it, and each carries
an honest "In ncad" status box and authoritative source citations. The concept lives here; the
capability lives in the [ncad manual](../ncad/index.md).

!!! note "In ncad"
    Each concept page ends with a status box like this one, drawn from the taxonomy: whether the
    concept is **Available**, **Partial**, **WIP**, **Planned**, or **Not-in-scope**, with an
    honest note and links into the ncad reference.
